## 🌐 Supported Indian Languages

This project uses **Tesseract OCR**, which supports multiple Indian languages.

### 📝 Available Language Codes

| Language  | Code  |
| --------- | ----- |
| Hindi     | `hin` |
| Punjabi   | `pan` |
| Gujarati  | `guj` |
| Tamil     | `tam` |
| Telugu    | `tel` |
| Bengali   | `ben` |
| Marathi   | `mar` |
| Kannada   | `kan` |
| Malayalam | `mal` |
| Sanskrit  | `san` |

---

### ⚙️ Usage

#### Single Language

```python
lang = "hin"
```
